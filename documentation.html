myFlix API

This API is designed to provide a user with an interface for discovering movie information. The API is currently hosted on a local server. The user will be able to search for information regarding movies, directors, and genres. Not only will users be able to seach for movie information, but they will also be able to create their own profiles. These profiles allow the user to save information about their favorite movies so they may log in to an account and reference what they saved.
